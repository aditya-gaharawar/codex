# @wsai/code-responses-api-proxy

<p align="center"><code>npm i -g @wsai/code-responses-api-proxy</code> to install <code>codex-responses-api-proxy</code></p>

This package distributes the prebuilt [WSAI CODE Responses API proxy binary](https://github.com/wsai/wsai-code/tree/main/wsai-code-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/codex-responses-api-proxy.js --help
```

Refer to [`wsai-code-rs/responses-api-proxy/README.md`](https://github.com/wsai/wsai-code/blob/main/wsai-code-rs/responses-api-proxy/README.md) for detailed documentation.
